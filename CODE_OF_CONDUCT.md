# Code de conduite » C2P

Un espace pense pour demarrer ou consolider ton aventure dans le monde des langages de programmation.  
Ici, on apprend ensemble, on progresse pas a pas, et on transforme la curiosite en reelles competences.

---

## ▸ Nos missions

```
● Revoir et comprendre les fondamentaux — variables, boucles, fonctions...
● Pratiquer a travers des defis, quiz et projets guides
● S'entraider et progresser ensemble, peu importe le niveau
● Decouvrir les bases de C, Python, JavaScript, Java, et bien plus
```

---

## ▸ Ce que tu peux faire ici

```
● Poser tes questions sans gene
● Partager ce que tu apprends ou ce qui te bloque
● Soumettre ton code dans le bon dossier de session
● Encourager et inspirer les autres membres
```

---

## ▸ Ce qui est attendu de chaque membre

```
● Respecter le travail et le niveau de chacun
● Commenter son code pour aider les autres a comprendre
● Utiliser un langage clair et bienveillant dans les echanges
● Ne pas modifier le dossier d'un autre binome
● Citer ses sources si tu t'inspires d'un travail existant
```

---

## ▸ Ce qui n'est pas tolere

```
▸ Tout contenu hors programmation et apprentissage
▸ Les moqueries ou jugements sur le niveau d'un membre
▸ Le plagiat sans mention de la source
▸ Les modifications non autorisees dans le repo
▸ Tout comportement qui nuit a l'ambiance d'apprentissage
```

Tout contenu non conforme pourra etre retire par l'equipe de moderation.

---

## ▸ L'esprit C2P

```
Tu n'es pas seul.
Ici, chaque ligne de code est un pas de plus vers ton avenir tech.
Tout ce qui est partage a un seul but : apprendre a coder.
Modules, ressources, projets, entraide — tout est aligne sur cette vision.
```

> "Les bases solides d'aujourd'hui sont les succes de demain."
